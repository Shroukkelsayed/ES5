A simple backend project built with (Node.js using ES5 syntax).
The project follows a (modular architecture) where database logic, controllers, and services are separated for better organization and scalability.

---
Project Structure

```
src/
 ├── DB/        # database connection and models
 ├── modules/   # feature modules (auth, user, etc.)
 └── app.controller.js  # main application controller

index.js              # application entry point
package.json          # project dependencies
package-lock.json     # dependency lock file
```

---
Run the application:

```
npm run dev
```

---
Technologies

* Node.js
* JavaScript (ES5)
* Modular backend architecture

---
Purpose

This project was created to practice building a clean and scalable backend folder structure using ES5 in Node.js.
